About JJDevHub

architecture Architecture Clean Architecture + DDD + CQRS

JJDevHub is built following Domain-Driven Design principles with CQRS pattern. PostgreSQL serves as the write store, MongoDB as the read store, and Kafka handles event streaming between them.
